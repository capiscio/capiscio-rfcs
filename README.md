## RFC Process

1. **Draft:** Open a PR with a new RFC in `rfcs/`
2. **Review:** Discussion happens in the PR
3. **Approved:** Merged to main
4. **Superseded:** Noted in the RFC header with pointer to replacement

## RFC Numbering
